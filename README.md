Adi Jaya Berkah - Furniture Management System

**Deskripsi Proyek**
Sistem Informasi Manajemen CV Adi Jaya Berkah berbasis PHP Native dan MySQL. Sistem ini dirancang untuk mendigitalisasi alur bisnis perusahaan manufaktur furniture kayu yang berlokasi di Jakarta Pusat. Memudahkan pemantauan stok bahan baku, proses produksi, hingga distribusi barang jadi ke pelanggan secara terorganisir.

**Fitur Utama**
* Dashboard Produksi: Visualisasi total pelanggan, bahan baku, barang jadi, dan jumlah pegawai secara real-time.
* Inventory Manufaktur: Manajemen stok kayu (bahan baku) dan furniture siap jual secara mendetail.
* Supply Chain: Kelola data supplier dan riwayat pembelian bahan baku kayu untuk kelancaran produksi.
* Point of Sales (POS): Pencatatan penjualan furniture disertai dengan fitur nota otomatis.
* Manajemen SDM: Pendataan pegawai yang terlibat dalam operasional perusahaan.

**Tech Stack**
* Language: PHP Native
* Database: MySQL
* Frontend: Bootstrap 5 & Custom CSS
